Testes feitos com o sensores de som de arduino, a fim de analisar a viabilidade funcional para a possivel montagem de um decibelímetro.

Sensores: 
-ky-037
-ky-038

Componentes:
-Arduino UNO
-Jumpers
-Leds
-Protoboard
